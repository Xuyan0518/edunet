# Admin Weekly Feedback Review Workflow

Deferred stakeholder workflow proposal:

1. Teacher saves weekly feedback as a draft.
2. Admin reviews the draft before parents can see it.
3. Admin can approve and publish the feedback.
4. Parent sees only published weekly feedback.
5. Parent notification is sent on publish, not on teacher draft save.

Suggested first version:

- Add weekly feedback status: `draft` and `published`.
- Teacher save creates or updates `draft`.
- Admin dashboard shows feedback awaiting approval.
- Admin publish makes the feedback parent-visible.
- Skip revision comments until stakeholders confirm the workflow.
